# 📰 News Briefing Skill

一个基于 Claude 的每日新闻简报 Skill，自动聚合来自全球主流媒体和科技企业的最新资讯，以结构化的中文简报形式呈现。

## 功能介绍

### 覆盖来源

| 类型 | 来源 |
|------|------|
| 国际媒体 | BBC、Reuters、AP、CNN、NYTimes、Al Jazeera |
| 中国媒体 | 新华社、南华早报、新浪财经、第一财经 |
| 中国科技企业 | 字节跳动 / TikTok、腾讯 / WeChat、小红书 |
| 海外科技企业 | 英伟达、微软、Meta、Google、苹果、OpenAI |

### 内容分类

- 🌍 **国际新闻** — 重大政治、外交、冲突、人道主义事件
- 💡 **科技 & AI** — 模型发布、产品更新、行业突破
- 🏢 **中国互联网企业动态** — 字节、腾讯、小红书等企业的最新动向
- 🌐 **海外科技企业动态** — 英伟达、微软、Meta、Google、苹果等
- 💰 **财经 & 市场** — 全球经济、市场行情、宏观政策

### 简报格式

每条新闻包含：
- **三句话概括**：用三句话交代清楚事件的核心事实、背景进展、影响走向
- **摘要段落**：补充关键细节和上下文，帮助读者深入理解
- **来源标注**：每条新闻标注原始媒体来源，底部附可点击链接

示例输出：

```
# 📰 每日简报 · 2026年04月22日 星期三

## 🌍 国际新闻

### 特朗普宣布延长对伊朗停火，和谈前景仍不明朗
美国总统特朗普宣布将无限期延长与伊朗的停火协议，以配合巴基斯坦斡旋的第二轮和平谈判。
这是他在两周内第二次在关键时刻撤回升级威胁，显示双方均有意避免全面战争。
然而，伊朗外交部发言人表示"目前尚无下一轮谈判计划"，和谈前景依然模糊。
> **摘要**：伊美冲突进入停火延续阶段……
> 来源：BBC、CNN、Al Jazeera

...

## 📌 今日关键词
伊朗停火延长、Meta Muse Spark、Tim Cook卸任、字节跳动净利下滑、IMF下调增速
```

## 安装方式

### 前提条件

- 安装并登录 [Claude Code](https://claude.ai/code) 或支持 Skill 的 Claude 客户端

### 安装步骤

1. 克隆或下载本仓库
2. 将 `news-briefing/` 文件夹复制到你的 Claude Skills 目录：
   ```bash
   cp -r news-briefing/ ~/.claude/skills/
   ```
3. 重启 Claude，Skill 即自动加载

## 使用方式

### 手动触发

在 Claude 对话中，直接说：

```
给我今天的新闻简报
```

```
今天发生了什么大事？
```

```
morning briefing
```

Claude 会自动识别意图并执行简报生成流程。

### 定时自动推送（每天 11 点）

通过 Claude Code 创建定时任务，每天上午 11 点自动生成简报并推送通知：

1. 在 Claude 中运行：
   ```
   帮我设置每天 11 点自动运行 news-briefing 简报
   ```
2. 或通过 Claude Code CLI 手动创建定时任务，任务文件参考 `scheduled-task-prompt.md`

定时任务创建后，可在 Claude 客户端侧边栏的 **Scheduled** 区域管理。

> **建议**：首次使用前点击"Run now"手动运行一次，完成 WebSearch / WebFetch 工具权限授权，避免定时任务因权限提示中断。

## 技术实现

- **数据获取**：通过 BBC RSS Feed（直接抓取）+ WebSearch（覆盖其他媒体和企业新闻）并行获取，确保时效性
- **内容处理**：自动过滤 24 小时外的内容，去重合并同一事件的多来源报道
- **语言**：全程简体中文输出，公司名 / 人名保留英文原名

## 自定义

如需修改覆盖的媒体来源、企业名单或输出格式，直接编辑 `SKILL.md` 中对应的搜索查询列表和输出模板即可。
